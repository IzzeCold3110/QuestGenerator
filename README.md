# QuestGenerator for Android(Unity)

## Description
- Quest-Editor for
  - title
  - description(short) 
  - description
- Outputs quests-<timestamp>.json to /mnt/sdcard/Download

## Install
- Download apk-file
- allow write-permission to SDCard
